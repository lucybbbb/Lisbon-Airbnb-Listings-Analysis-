<h1>Lisbon Airbnb Listings Analysis</h1>

<p>
This project analyzes Airbnb listings in Lisbon to understand pricing trends, neighbourhood performance, host behaviour, and customer reviews using data analysis and visualization tools.
</p>

<hr>

<h2> Project Objective</h2>
<p>
The goal of this project is to explore Airbnb listings in Lisbon and answer key business questions such as:
</p>

<ul>
  <li>Which neighbourhoods are the most expensive?</li>
  <li>How does room type affect price?</li>
  <li>What drives higher ratings?</li>
  <li>Which areas have the most listings?</li>
</ul>

<hr>

<h2>Tools & Technologies</h2>

<ul>
  <li>Excel / Power Query – Data Cleaning</li>
  <li>SQL – Data Analysis</li>
  <li>Power BI – Data Visualization</li>
  <li>Git & GitHub – Version Control</li>
</ul>

<h2>Dataset</h2>

<p>
The dataset contains Airbnb listings in Lisbon with the following key fields:
</p>

<ul>
  <li>id</li>
  <li>host_id</li>
  <li>neighbourhood</li>
  <li>room_type</li>
  <li>price</li>
  <li>bedrooms</li>
  <li>minimum_nights</li>
  <li>maximum_nights</li>
  <li>review_scores_rating</li>
  <li>number_of_reviews_ltm</li>
</ul>

<hr>

<h2> Data Cleaning Steps</h2>

<ul>
  <li>Removed duplicate records</li>
  <li>Handled missing values</li>
  <li>Converted price to numeric format</li>
  <li>Standardized column names</li>
  <li>Validated data ranges (price, bedrooms, ratings)</li>
</ul>

<h2> Key Insights</h2>

<ul>
  <li>Central neighbourhoods tend to have higher prices</li>
  <li>Entire homes are significantly more expensive than private rooms</li>
  <li>Higher availability does not always mean lower price</li>
</ul>
